## 🚀 Deployment (GitHub Pages)

1. Push all files to your GitHub repository.
2. Go to **Settings** → **Pages** → **Source** → Select `main` branch & `/ (root)` folder.
3. Wait ~1-2 minutes. Your game will be live at:
   `https://<your-username>.github.io/<your-repo-name>/` 🔁
4. On mobile: Open the link → Tap browser menu → **"Add to Home Screen"** for native PWA experience.

> 💡 **Tip**: All asset filenames are case-sensitive. Ensure they match exactly as listed above.
